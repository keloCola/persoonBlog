##  前端与链表：使用链表指针获取 JSON 的节点值 

```js
const json = {
    a: { b: { c: 1 } },
    d: { e: 2 },
};

const path = ['a', 'b', 'c'];
// 新建指针p 指向json
let p = json;
// 类似于遍历链表
path.forEach((k) => {
    p=p[k];
});

```
