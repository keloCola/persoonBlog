### 解题

```js
var deleteDuplicates = function(head) {
  let p = head;
    while (p && p.next) {
      if (p.val === p.next.val) {
        p.next = p.next.next
      }else{
          p = p.next
      }
    }
    return head
  };

```

时间复杂度是O(n) 空间复杂度是O(1)
