## LeetCode：206.反转链表

反转一个单链表。

**示例**

**输入** 1->2->3->4->5->NULL

**输出** 5->4->3->2->1->NULL

```js
var reverseList = function(head) {
  
};

```

```js
var reverseList = function(head) {
  let p1 = head
  let p2 = null
  while(p1){
      const tmp = p1.next
      p1.next=p2
      p2= p1
      p1=tmp
  }
  return p2
};

```

时间复杂度 O（n）

空间复杂度O(1)
